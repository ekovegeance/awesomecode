## How to use
1. Open VSCode.
2. Navigate to `File` > `Preferences` > `Settings` (or press `Cmd + ,`).
3. In the top-right corner, click the **Open Settings (JSON)** icon.
4. Copy the `settings.json` provided in this repository and paste it into your VSCode **settings.json** file.
5. Save the file and restart VSCode to apply the changes.

**Enjoy your awesome coding setup! 🚀**
